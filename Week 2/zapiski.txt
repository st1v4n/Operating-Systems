Inode -> мета данни за даден файл
ls -l -i -> показва файловете + техния Inode id
stat <file> -> показва статистики за файл
hardlink -> Второ име за даден файл
ln <path> <file_name> -> прави хард линк (и нов файл) към файл със същия Inode
(*) -> hardlink-ове не могат да сочат към директории или файлове на друга файлова система
ls -li <file> -> брой имена към файл
symlink -> файл, който в началото си съдържа път към друг файл, нямат един и същ Inode
ln -s <path> <file_name> -> създава symlink
(*) -> местенето чупи symlink-а
cat <file> -> четем съдържанието на файл
realpath <file> -> дава път, който е дерефериран (спрямо линковете)
basename <file> -> дава име на файла, без да следва линковете
dirname <file> -> дава името на директорията, без да следва линковете
readlink <file> -> следва само първия symlink
stat -L <file> -> следва ликновете и дава статистики
stat -c '%<нещо>' -> дава информация за конкретно нещо
chmod <права> <file> -> задава права за даден файл
find <globals> <directory> <flags> <actions> -> търси по критерий и извършва действия
