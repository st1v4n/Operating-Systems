pwd -> изписва текущата директория по абсолютен път
ls -> показва всички файлове в текущата директория
ls -l -> показва повече информация за файловете, където -l е ФЛАГ
ls -a -> показва скрити файлове
ls ~/example -> показва всички файлове в съответната директория, където ~ е home директорията
tree -> принтира директориите като дърво, започвайки от текущата директория
touch <име> -> създаване на файл в текущата директория, може да се даде и път
touch <име1> <име2> ... -> създаване на много файлове
cp <път откъде> <път накъде> -> копиране на файл
. -> текущата директория
.. -> предишната директория
mkdir <име> -> създана празна директория
mv <път откъде> <път накъде> -> премества файл
rmdir <име> -> трие празна директория
rm <име> -> трие файл
rm -r <име> -> трие непразна директория
mkdir -p <път> -> създава всички нови директории по пътя
man <команда> -> показва информация за нея, навигира се със стрелките
cd <път> -> отиване в дадена директория
